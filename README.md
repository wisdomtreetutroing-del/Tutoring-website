## 🎯 CUSTOMIZATION GUIDE

### Change Colors
Edit `styles.css` lines 8-18 and 21-31:
```css
[data-theme="light"] {
    --primary: #D4AF37;  /* Your golden color */
    --secondary: #FFD700; /* Your accent */
}
```

### Change Fonts
Edit HTML `<head>`:
```html
<link href="https://fonts.googleapis.com/css2?family=YourFont&display=swap">
```

Then update CSS:
```css
body { font-family: 'YourFont', sans-serif; }
```

### Add More Sections
1. Copy existing section in HTML
2. Edit content
3. Style in CSS
4. Add to admin panel (optional)
